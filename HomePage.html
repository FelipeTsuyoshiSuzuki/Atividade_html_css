Aplicativo para compras de veículos

O que o aplicativo City Car Service Cente oferece para você?

1. Ele organiza seus dados sobre cada veículo, como modelo, marca, ano, etc.

2. Ele organiza as informações sobre os clientes como moeda usada na transação, país de origem, e-mail, etc.

3. Possui uma interface amigavel para o usuário
